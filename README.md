# 说明文档
## 支持

1. 基本数据类型、结构体类型、结构体的指针类型、切片类型（元素可以是：基本数据类型、结构体类型、结构体的指针类型）

## 不支持

1. `bool`、结构体、结构体指针类型、切片不支持默认值
2. 切片的元素是结构体指针类型且元素是`nil`时，该元素跳过任何处理

## 注意

对于切片，代码会始终保证不是nil切片

对于结构体指针类型，代码会始终保证不是nil


## 需要测试的功能
### 配置文件解析测试

目的是测试程序是否能正常解析配置文件到结构体

测试场景包括但不限于：
1. 各种类型的字段是否解析正常
2. 结构体的字段是结构体的情况下各种类型的字段是否解析正常
3. 结构体的字段是切片的情况下各种类型的字段是否解析正常
4. 结构体的字段是切片且切片的元素是结构体的情况下各种类型的字段是否解析正常
5. 结构体的字段是切片且切片的元素是结构体的指针类型的情况下各种类型的字段是否解析正常
6. 结构体的字段是结构体指针类型且不为nil
7. 结构体的字段是结构体指针类型且为nil
8. 结构体的字段是切片类型且不为nil
9. 结构体的字段是切片类型且为nil

### 配置文件默认值测试

目的是测试默认值是否能正常赋值

测试场景类同上面的“配置文件解析测试”

### 其他测试

目的是测试程序是否满足需求

测试的需求有：
1. 支持的数据类型：基本数据类型、结构体类型、结构体的指针类型、切片类型（元素可以是：基本数据类型、结构体类型、结构体的指针类型）
2. 对于切片，代码会始终保证不是nil切片
3. 对于结构体指针类型，代码会始终保证不是nil
4. `bool`、结构体、结构体指针类型、切片不支持默认值，但其内部（结构体的字段、结构体指针指向的结构体的字段、切片的结构体元素）不受影响
